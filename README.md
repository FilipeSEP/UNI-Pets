# UNI-Pets 🐾 | E-commerce Platform for Pet Shops

UNI-Pets is a full-stack web application designed to streamline the shopping experience for pet owners and management for shop owners. This project demonstrates my ability to build a scalable, responsive, and functional e-commerce solution.

## 🚀 Features

- **Functional Shopping Cart:** Features local browser data persistence, allowing users to keep their items even after refreshing the page.
- **User Authentication:** Integrated login system for secure access.
- **Dynamic Product Display:** Interactive catalog with "About the Store" section.
- **Responsive Design:** Optimized for all screen sizes (Mobile, Tablet, and Desktop).
- **Clean Architecture:** Built with a focus on maintainability and code readability.

## 🛠️ Tech Stack

- **Backend:** Java (Logic and Server-side processing)
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Storage:** LocalStorage for browser-side data persistence
- **Version Control:** Git

## 📈 What I Learned

During the development of UNI-Pets, I improved my skills in:
- Handling **asynchronous operations** in JavaScript.
- Implementing **Data Persistence** without complex databases (using browser storage).
- Designing **Responsive Layouts** from scratch using modern CSS techniques.
